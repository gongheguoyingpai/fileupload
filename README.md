FileUpload use plupLoad and Java
===================================

我们都知道HTML有一个file控件，长的奇丑
，并且如果选择多个文件时就不是丑的问题
了，只是显示一个选择了多少个文件的数字
根本不知道到底选择的是哪些文件，不能忍
但是，每次重复写是不是很无语，所以，我
写了这个小破东西，你可以很轻松的把它加
如到你的web项目里面去.

你需要做的只是按照我的目录结构，当然也
可以是你自己的目录结构，稍微改一下引入
js和css的路径就可以啦。我是在plupLoad最
简单的上传样式上做的改进，使得可以删掉选
择的文件，并且可以清空选择的文件列表。

你会发现这里面还包括了将文件名重命名为用
SHA-1生成的签名作为文件名从而避免了文件名
重复的问题。
